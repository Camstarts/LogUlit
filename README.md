# LogUlit
Android library for working whit Log file android SDK.


## Build & Run
1. Clone the repo
2. `./gradlew fatJar`
3. `compile 'com.github.Camstarts:LogUlit:v0.1.0`


allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}

dependencies {
	        compile 'com.github.Camstarts:LogUlit:v0.1.0'
	}
